# Python_API_Challenge
Analysis of equator weather data utilizing API calls, Python Requests, and JSON traversals

**Weather Analysis**

Created a python script to visualize the weather of 500+ cities across the world utilizing the OpenWeatherMap API. Then plotted the results based on different cities and weather data (humidity, wind speed, temperature etc). 

**Vacation Analysis**
Using the list of 500 cities, utilized jupyte-gmaps and Google Places API to create various heat maps based on the cities data called in step 1.
